# vt-parcels
![GitHub](https://img.shields.io/github/license/wasac/vt-parcels)

parcels vector tiles data in Rwanda

## Usage

- create vector tiles
```
npm i
npm run create
```

- extract pbf from mbtiles
```
npm run extract
```

- deploy to gh-pages
```
npm run deploy
```
